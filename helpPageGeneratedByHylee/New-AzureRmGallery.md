# New-AzureRmGallery

## SYNOPSIS
Create a gallery.

## SYNTAX

```
New-AzureRmGallery [-ResourceGroupName] <String> [-GalleryName] <String> [-Gallery] <PSGallery> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Create a gallery.

## EXAMPLES

### Example 1
```powershell
PS C:\> New-AzureRmGallery -ResourceGroupName $rgname -GalleryName $galleryName -Gallery $galleryObject
```

Create a gallery.
